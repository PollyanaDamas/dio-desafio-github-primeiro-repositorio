A sigla SHA significa (Secure Hash Algorithm) Tradução: Algoritmo de Hash Seguro.
É um conjunto de funções hash criptógrafadas progetadas pela NSA 
( Agência De Segurança Nacional dos EUA).
A encriptação gera um conjunto de caracteres identificador, são 40 dígitos.
SHA1 é uma forma curta de representar um arquivo.
